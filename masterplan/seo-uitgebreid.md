## 5. Technical SEO Deep Dive {#technical-seo}

### 5.1 Core Web Vitals Optimalisatie

**Largest Contentful Paint (LCP) < 2.5s**

**Acties:**
- [ ] Image lazy loading (al geÃ¯mplementeerd?)
- [ ] Gebruik WebP formaat voor alle images
- [ ] CDN implementeren (Cloudflare, BunnyCDN)
- [ ] Font preloading:
  ```html
  <link rel="preload" href="/fonts/main.woff2" as="font" type="font/woff2" crossorigin>
  ```
- [ ] Critical CSS inlinen
- [ ] Code splitting per route (React lazy loading al gebruikt)

**First Input Delay (FID) < 100ms**
- [ ] Minimize JavaScript execution time
- [ ] Use Web Workers voor zware taken
- [ ] Debounce/throttle scroll & resize events

**Cumulative Layout Shift (CLS) < 0.1**
- [ ] Width & height op ALLE images
- [ ] Reserveer ruimte voor ads/embeds
- [ ] Geen content shifts bij font loading (font-display: swap)
